# Pineapple II Ports

## External Ports

### Power

#### Type

EIAJ Class-3 (DC 6.3V to 10.5V, 2A)

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | V+       |
| 2        | GND      |

### GeekPort II

#### Type

Half-pitch 14-pin Amphenol Connector

#### Pinout

| Pin num. | Function     |
| -------- | ------------ |
| 1        | V+           |
| 2        | GND          |
| 3        | USB D+       |
| 4        | USB D-       |
| 5        | USB Power    |
| 6        | Cable detect |
| 7        | Vcc          |
| 8        | Sensor A     |
| 9        | Sensor B     |
| 10       | Sensor C     |
| 11       | Sensor D     |
| 12       | I2C SDA      |
| 13       | I2C SCL      |
| 14       | GND          |

### MIDI OUT+

#### Type

DIN 5-pin

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | Vcc      |
| 2        | GND      |
| 3        | Raw TX   |
| 4        | Send     |
| 5        | Return   |

### MIDI IN+

#### Type

DIN 5-pin

#### Pinout

| Pin num. | Function  |
| -------- | --------- |
| 1        | Contact A |
| 2        | NC        |
| 3        | Contact B |
| 4        | Anode     |
| 5        | Kathode   |

### Sensor Port

#### Type

Mini DIN 4-pin

#### Pinout

| Pin num. | Function     |
| -------- | ------------ |
| 1        | Vcc          |
| 2        | GND          |
| 3        | Sensor A     |
| 4        | Cable detect |

### Relay Port

#### Type

Mini phone jack.

#### Pinout

| Pin num. | Function  |
| -------- | --------- |
| 1        | Contact A |
| 2        | Contact B |

## Internal Ports

### XMIDI

#### Type

JST PH connector (2mm-pitch)

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | TX+      |
| 2        | TX-      |
| 3        | RX+      |
| 4        | RX-      |

### Driver

#### Type

2x3 Pinheader

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | GPIO0    |
| 2        | GPIO1    |
| 3        | GPIO2    |
| 4        | SPI MISO |
| 5        | SPI MOSI |
| 6        | SPI SLCK |

### LED

#### Type

1x4 Pinheader

#### Pinout

| Pin num. | Function    |
| -------- | ----------- |
| 1        | TXA (GPIO0) |
| 2        | TXK         |
| 3        | RXA (GPIO1) |
| 4        | RXK         |

### Sensor XY

#### Type

1x2 Pinheader

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | Sensor X |
| 2        | Sensor Y |


### Front

#### Type

2x4 Pinheader

#### Pinout

| Pin num. | Function |
| -------- | -------- |
| 1        | V+       |
| 2        | GND      |
| 3        | V+       |
| 4        | GND      |
| 5        | RST      |
| 6        | GND      |
| 7        | LEDA     |
| 8        | LEDK     |

### Back

#### Type

10-pin Harting (MIL) connector

#### Pinout

| Pin num. | Function        |
| -------- | --------------- |
| 1        | Vcc             |
| 2        | GND             |
| 3        | Raw TX          |
| 4        | MIDI OUT Send   |
| 5        | MIDI OUT Return |
| 6        | Contact A       |
| 7        | GND             |
| 8        | Contact B       |
| 9        | MIDI IN Anode   |
| 10       | MIDI IN Kathode |
